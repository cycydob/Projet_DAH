# Projet_DAH
ce projet porte sur la détection des maladies des plantes
Projet 9 : Classification d’images de plantes (santé des feuilles)
  Le dataset contient des images de feuilles saines ou atteintes de différentes maladies. 
* Idée de projet :
  - 1. Détection de maladies via CNN.
  -  2. Dashboard simple pour charger une image et avoir le diagnostic.
        Dataset Lien : ____https://www.kaggle.com/datasets/vipoooool/new-plant-diseases-dataset____
